# houker的构思

## 基础功能构思

   1. 自主寻路（可以没电返仓）

   2. 可以为电脑做外设（蓝牙，WiFi，有线）

      + 摄像头（支持人脸居中）
      + 音量，亮度调节器
      + 应用切换
      + 眼动追踪
      + 可当做普通linux开发板使用

   3. 拥有语音助手，可当做控制中枢
